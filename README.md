# Odin Recipes

A basic recipe website built as part of [The Odin Project](https://www.theodinproject.com/lessons/foundations-recipes) Foundations curriculum.

## About

This project is a pure HTML website with no styling. The goal is to practice core HTML skills — structuring pages, working with headings, lists, links, and images — before moving on to CSS.

The site consists of a main index page linking to individual recipe pages, each with a description, ingredients list, and step-by-step instructions.

## Skills Demonstrated

- HTML document structure and boilerplate
- Headings, paragraphs and text formatting
- Unordered and ordered lists
- Linking between pages with anchor tags
- Embedding images
- Organizing a project with directories
- Git workflow — committing early and often with meaningful commit messages
- Deploying a static site via GitHub Pages

## Getting Started

Clone the repository and open `index.html` in your browser:

```bash
git clone git@github.com:your-username/odin-recipes.git
cd odin-recipes
open index.html
```

No dependencies or build steps required.

## What I Learned

This project introduced the fundamental building blocks of HTML — how to structure a document, navigate between pages using relative links, and present content clearly using semantic elements like lists and headings. It also reinforced a solid Git workflow habit of committing changes incrementally as the project grows.
